Acerca de

Plataforma web educativa para ortografía con unidades, ejercicios interactivos y reportes automáticos.

¿Qué es el Recomendador Inteligente de Contenidos?

Es una plataforma desarrollada con Django y MySQL para apoyar la enseñanza de ortografía. Organiza unidades (Acentuación, Puntuación, Mayúsculas y Reglas de las letras) con ejercicios interactivos, corrección inmediata, almacenamiento de intentos, cálculo de porcentajes y reporte individual con recomendaciones.

Django MySQL Evaluación automática
Misión

Facilitar el aprendizaje autónomo con ejercicios interactivos, feedback inmediato y recomendaciones personalizadas con IA.

Visión

Ser una referencia digital para el estudio de lengua, integrando evaluaciones y análisis de desempeño para decisiones informadas.

¿A quién beneficia?
Estudiantes

Acompañamiento individual, ejercicios adaptados, explicaciones automáticas y calendario con avances.

Docentes

Informe consolidado por unidad, dominio alcanzado y generación de PDF para seguimiento.

Instituciones y familias

Datos objetivos del rendimiento para evaluar estrategias y ofrecer soporte adicional.

¿Cómo utilizar el sistema?
  1. Acceso: iniciar sesión con usuario y contraseña.
  2. Guía de aprendizaje: unidades con PDF de teoría y ejercicios interactivos con corrección y explicación.
  3. Biblioteca: buscar y descargar libros en PDF para estudio autónomo.
  4. Progreso y resultados: se guardan intentos, se calcula el porcentaje y se puede descargar un reporte.
  5. Calendario de actividades: registra las prácticas por fecha con hora, título y descripción.
  6. Perfil: visualización de datos personales cargados en la plataforma.
  7. Rol docente: panel de evaluaciones y generación de informes PDF.
